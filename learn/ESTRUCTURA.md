## 🚀 Cómo Usar

### 1. Ejecutar Ejemplos
```bash
# Ir a cualquier carpeta
cd learn/01_data_type

# Ejecutar con tsx (recomendado)
npx tsx examples.ts

# O compilar y ejecutar
tsc examples.ts
node examples.js
```
