# 📚 TypeScript Learning Path

A comprehensive, hands-on guide to mastering TypeScript from fundamentals to advanced concepts.

## 🎯 Learning Structure

This course is organized into progressive modules, each building upon the previous ones:

### **Fundamentals (Beginner)**
- **01_data_type** - Primitive types, arrays, tuples, enums
- **02_functions-and-objects** - Function signatures, object types, optional properties
- **03_interfaces-and-types** - Interface vs Type, extending interfaces, intersection types

### **Core Concepts (Intermediate)**
- **04_classes-and-oop** - Classes, inheritance, access modifiers, abstract classes
- **05_generics** - Generic functions, classes, constraints, default types
- **06_advanced-types** - Union, intersection, literal types, type aliases
- **07_modules-and-namespaces** - Import/export, module resolution, namespaces

### **Advanced Features**
- **08_decorators** - Class, method, property, parameter decorators
- **09_utility-types** - Partial, Required, Pick, Omit, Record, etc.
- **10_async-patterns** - Promises, async/await, typing callbacks
- **11_type-guards** - typeof, instanceof, custom type guards, discriminated unions

### **Expert Level**
- **12_conditional-types** - Conditional types, infer keyword, distributive types
- **13_mapped-types** - Creating custom utility types, template literal types
- **14_best-practices** - Project structure, tsconfig, linting, testing

## 📖 How to Use This Guide

1. **Follow the numbered order** - Each module builds on previous knowledge
2. **Read the README.md** in each folder for theory and explanations
3. **Study the examples** - Each folder contains practical code examples
4. **Complete exercises** - Practice with the provided exercises
5. **Run the code** - Compile and execute TypeScript files to see results

## 🚀 Getting Started

```bash
# Make sure TypeScript is installed
npm install -g typescript

# Compile a TypeScript file
tsc filename.ts

# Watch mode for automatic compilation
tsc --watch filename.ts

# Run directly with ts-node
npx ts-node filename.ts
```

## 🎓 Learning Tips

- **Practice daily** - Even 30 minutes of consistent practice is better than sporadic long sessions
- **Type everything manually** - Don't copy-paste, typing helps muscle memory
- **Experiment** - Modify examples to see what happens
- **Build projects** - Apply what you learn in real projects
- **Read compiler errors** - TypeScript errors are educational
